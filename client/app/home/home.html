Tucson PD Data

80,000 Incident reports over 3 months - in 2 minutes.

Your City, Visualized

This project aims to utilize modern web technologies to visualize open source city data. We used incident report data from the Tucson Police Department to create a timelapse view. This allows for patterns in the data to be seen in an intuitive way.

Built by: Tim Gallant, Timmy Garrabrant, Hasanain Jamal, Sean Stephens
